# ankietaplus-blog

SEO content hub for AnkietaPlus (B2B + B2C acquisition).

## Includes
- Landing pages: `content/landings/*.md` (12)
- Blog posts: `content/blog/*.md` (16)
- SEO strategy docs: `docs/seo/*.md`
- LLM guidance: `public/llms.txt`
- JSON-LD, robots, sitemap

## Dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run lint
npm run build
```
