# GSC + Monitoring Plan (operacyjny)

## Konfiguracja
1. Dodać property domenowe i URL prefix dla bloga.
2. Podpiąć sitemap: `https://ankietaplus-blog.vercel.app/sitemap.xml`.
3. Włączyć alerty coverage + manual actions.

## Dashboard tygodniowy
- kliknięcia organiczne
- wyświetlenia
- CTR
- średnia pozycja
- TOP 20 stron i TOP 50 zapytań

## Iteracja CTR (co tydzień)
- wybierz 10 URL z dużymi impresjami i CTR <2.5%
- przepisz title + description
- porównaj 14 dni before/after

## Reguła update contentu
- jeśli URL spada >20% kliknięć m/m -> update treści + FAQ + internal links
